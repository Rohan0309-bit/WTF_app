# **App Name**: Well Trained Freak

## Core Features:

- Authentication: Authenticate users via email/password and Google OAuth using Firebase Authentication with session synchronization.
- Profile Setup: Enable users to set up profiles specifying if they are an athlete or general user, and collect data such as name, age, gender, weight, and height, storing data in Firebase with local storage fallback.
- Calculators: Include BMI and Calorie calculators for users to track their health metrics.
- Athlete Workout Plans: AI tool to Generate sport-specific workout plans tailored to male and female athletes for both home and gym, supporting sports like Cricket, Badminton, Football, Volleyball, Basketball, Table Tennis, and Calisthenics.
- Daily Workout View: Provide daily workout view with exercises detailed by sets, reps, rest time, and animated demonstrations (GIFs/videos) alongside integrated timers for rest periods and voice cues for exercise changes.
- Nutrition Plans: Enable full-day professional meal plans for cutting, bulking, and lean muscle gain, tailored for each sport, stored as json data.
- Posing Tutorials: Exclusive posing tutorials specifically for physique presentation.

## Style Guidelines:

- Primary color: Vibrant green (#00A36C) to represent health and energy, fitting the 'freak mode' vibe.
- Background: Light pastel green (#E0F2E9) to provide a calming, yet energetic backdrop.
- Accent Color: Yellow-green (#9ACD32) for interactive elements and highlights, adding a lively contrast.
- Headlines: 'Poppins' (sans-serif) for a modern and readable aesthetic.
- Body: 'PT Sans' (sans-serif) ensuring comfortable readability throughout the app.
- Use emojis where possible for exercise types, muscle groups, and injury zones (🦵, 🔙, 💪) to enhance visual communication.
- Implement subtle sports-themed motion effects for athlete users and smooth transitions for general users to enrich engagement and navigation.
